# customer-churn-analysis-and-prediction

## Overview
Analyzed telecom customer churn patterns and built a machine learning model to predict customer churn.

## Dataset
- 7043 customer records
- Telecom customer dataset

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Findings
- Month-to-month customers have the highest churn.
- Customers with shorter tenure are more likely to churn.
- Higher monthly charges increase churn risk.
- Fiber optic customers show higher churn rates.

## Machine Learning
- Random Forest Classifier
- Accuracy: 79.8%

## Feature Importance
- Tenure Months
- Total Charges
- Monthly Charges
- CLTV

## Business Recommendations
- Improve customer retention programs.
- Encourage long-term contracts.
- Reduce churn among new customers.
